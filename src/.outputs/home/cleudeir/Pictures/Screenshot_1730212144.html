Código de Verificação

Descrição da tela

Tela de verificação de código para acesso a uma conta. Apresenta o logo da empresa UAU, o e-mail para o qual o código foi enviado e um campo para inserir o código recebido.

Regra de Negócio:

  • Lógica de Processamento: O usuário digita o código de verificação recebido por e-mail no campo "Digite o código". Ao clicar em "Enviar", o sistema valida o código. Se o código for válido, o usuário é redirecionado para a próxima etapa. Caso contrário, uma mensagem de erro é exibida.

  • Ações Condicionais: Se o código for válido, o usuário é redirecionado. Se o código for inválido ou o campo estiver vazio, exibe mensagem de erro.
