Observação :

  • Os campos com o ícone de cadeado permitem que o valor seja travado para reutilização em futuras transferências, facilitando o preenchimento para valores que são frequentemente reutilizados.

Regra de Negócio:

  • Lógica de Processamento: O usuário insere seu e-mail e senha nos campos correspondentes. Ao clicar em "Login", o sistema valida as credenciais. Se as credenciais forem válidas, o usuário é redirecionado para a tela principal do sistema. Caso contrário, uma mensagem de erro é exibida.
  • Ações Condicionais: Se o usuário clicar em "Cadastre-se", ele é redirecionado para a tela de cadastro. Se clicar em "Redefinir", ele é redirecionado para a tela de redefinição de senha.
