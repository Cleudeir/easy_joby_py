Tela de Login UAU

Descrição da tela

Tela de login para acesso ao sistema UAU. Apresenta campos para inserção de e-mail e senha, botão de login, e opções para cadastro e redefinição de senha.

Formulário :

Formulário de Login

Nome Tipo Validação Observação
Email Texto Email válido -
Senha Senha Mínimo de caracteres (não especificado) -

Observação :

  • Os campos com o ícone de cadeado permitem que o valor seja travado para reutilização em futuras transferências, facilitando o preenchimento para valores que são frequentemente reutilizados.

Regra de Negócio:

  • Lógica de Processamento: O sistema verifica se o e-mail e a senha inseridos correspondem a um usuário cadastrado no banco de dados.

  • Ações Condicionais:

    • Se o e-mail e a senha estiverem corretos, o usuário será redirecionado para a tela principal do sistema.
    • Se o e-mail ou a senha estiverem incorretos, será exibida uma mensagem de erro.
    • O botão "Cadastre-se" redireciona o usuário para a tela de cadastro.
    • O link "Esqueceu sua senha?" redireciona o usuário para a tela de redefinição de senha.
