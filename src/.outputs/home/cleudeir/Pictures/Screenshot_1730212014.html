Alterar Senha

Descrição da tela

Tela para alteração de senha, solicitando o email do usuário para envio de código de verificação.

Formulário :

Formulário de Alteração de Senha

Nome Tipo Validação Observação
Informe o seu e-mail Texto Email válido -

Regra de Negócio:

  • Lógica de Processamento: O usuário insere seu email no campo "Informe o seu e-mail". Ao clicar em "Enviar", o sistema valida o email. Se válido, envia um código de verificação para o email informado. Caso inválido, apresenta mensagem de erro.

  • Ações Condicionais: Se o email for válido, envia código de verificação e redireciona para próxima tela. Se o email for inválido, exibe mensagem de erro.
