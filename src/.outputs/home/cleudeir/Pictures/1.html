Nome da Tela

Análise de Dados Agrícolas

Descrição da Tela

A tela apresenta uma análise detalhada de dados agrícolas, fornecendo informações sobre a safra atual e o histórico de produção. Ela inclui gráficos e tabelas que permitem a visualização das tendências e padrões nos dados.

Campos Input/Busca/Filtro:

Nome da Coluna Tipo Validação Observação
Data Inicial Data Deve ser uma data válida Utilizada para definir a faixa de datas para análise
Data Final Data Deve ser uma data posterior à inicial Utilizada para definir a faixa de datas para análise
Cidade Texto Deve corresponder a uma cidade válida Utilizada para filtrar os dados por cidade

Cartões com Indicadores:

Nome da Coluna Tipo Validação Observação
Safra Atual Percentagem Deve ser um valor entre 0 e 100 Mostra a porcentagem de safra atual em relação ao ano anterior
Produção Histórica Unidades Deve ser um número válido Mostra a produção histórica de produtos agrícolas

Gráficos:

Nome da Coluna Tipo Eixo X Eixo Y
Safra Anual Linha Data Percentagem
Produção Por Cidade Barra Cidade Unidades

Lista:

Nome da Coluna Tipo Validação Observação
Produtos Agrícolas Texto Deve corresponder a um produto agrícola válido Mostra uma lista de produtos agrícolas

Formulário :

Não existente

Regra de Negócio

Lógica de Processamento:

  • Quando o usuário seleciona uma faixa de datas, o sistema atualiza automaticamente os gráficos e tabelas com os dados correspondentes.
  • Quando o usuário seleciona um produto agrícola, o sistema apresenta apenas os dados relacionados a esse produto.

Ações Condicionais:

  • O sistema não permite que o usuário selecione uma data inicial posterior à final.

Permissões:

  • Apenas usuários autorizados podem acessar esta tela.
