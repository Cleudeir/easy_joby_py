Descrição da tela

Tela inicial para clientes, exibindo serviços solicitados e finalizados.

Lista:

Nome Tipo Validação Observação
Número Número - Número de identificação do serviço.
Local Texto - Local do serviço.
Data Data Formato DD/MM/AAAA - HH:MM Data e hora do serviço.
Serviço Texto - Descrição do serviço realizado.
Status Texto Pagamento realizado, OS concluída Status do serviço.
Status Pagto. Texto Pago Status do pagamento do serviço.
Clique aqui para mais detalhes Link - Link que direciona para mais detalhes do serviço.

Botões:

Nome Tipo Ação
SOLICITADOS Botão Exibe os serviços solicitados.
FINALIZADOS Botão Exibe os serviços finalizados.
Solicitar Novo Serviço Botão Direciona para tela de solicitação de serviço

Regra de Negócio:

  • Lógica de Processamento: A tela exibe uma lista de serviços, divididos em "Solicitados" e "Finalizados", com detalhes como número, local, data, serviço, status e status de pagamento. O usuário pode clicar em "Clique aqui para mais detalhes" para visualizar informações adicionais sobre um serviço específico. O botão "Solicitar Novo Serviço" leva o usuário para uma nova tela para solicitar um novo serviço.

  • Ações Condicionais: A exibição da lista de serviços muda dependendo do botão selecionado ("Solicitados" ou "Finalizados"). Ao clicar em "Solicitar Novo Serviço", o aplicativo navega para uma nova tela.
