Observação :

  • Os campos com o ícone de cadeado permitem que o valor seja travado para reutilização em futuras transferências, facilitando o preenchimento para valores que são frequentemente reutilizados.

Regra de Negócio:

  • Lógica de Processamento: O usuário preenche todos os campos obrigatórios do formulário. Após clicar em "Cadastrar", o sistema valida os dados. Se os dados forem válidos, o usuário é cadastrado. Se os dados forem inválidos, o sistema exibe mensagens de erro.

  • Ações Condicionais: Se o usuário marcar as caixas de seleção "Concordo com as políticas de privacidade" e "Concordo com os termos de uso", o botão "Cadastrar" estará ativo. Caso contrário, o botão ficará desabilitado. Após cadastro, redirecionamento para tela de login ou outra tela definida.
