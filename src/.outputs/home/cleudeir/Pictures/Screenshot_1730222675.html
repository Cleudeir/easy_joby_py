Descrição da tela

Tela principal do aplicativo, exibindo os serviços disponíveis e aceitos, com detalhes de cada serviço.

Lista:

Nome Tipo Validação Observação
Número Texto - Número de identificação do serviço.
Cliente Texto - Nome do cliente.
Local Texto - Local de prestação do serviço.
Data Serviço Data Formato DD/MM/AAAA - HH:MM Data e hora agendadas para a execução do serviço.
Serviço Texto - Descrição do serviço prestado.
Status Texto - Status atual do serviço (Pagamento realizado, OS concluída).
Status Pagto. Texto - Status do pagamento (Pago).
Clique aqui... Botão - Botão para acessar mais detalhes do serviço.

Regra de Negócio:

  • Lógica de Processamento: A tela lista os serviços, agrupados por status (Disponíveis e Aceitos). Cada item da lista contém informações detalhadas sobre o serviço, como número, cliente, local, data, serviço prestado, status e status de pagamento. Ao clicar em "Clique aqui para mais detalhes", o usuário é redirecionado para uma tela com informações mais detalhadas sobre o serviço selecionado.

  • Ações Condicionais: A exibição dos serviços é condicionada ao status (Disponíveis ou Aceitos). A navegação para mais detalhes é disparada pelo clique no botão correspondente a cada serviço.
