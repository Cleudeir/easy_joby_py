Tela para inclusão de foto e legenda para um serviço.

Formulário :

Formulário de inclusão de foto

Nome Tipo Validação Observação
Descrição da foto Texto - -
Foto Imagem Tipo de arquivo, tamanho máximo -

Regra de Negócio:

  • Lógica de Processamento: O usuário preenche o campo "Descrição da foto" com um texto que descreve a imagem e seleciona uma imagem para o campo "Foto". Após preencher os campos, o usuário clica em "Enviar" para salvar as informações ou em "Voltar" para cancelar a operação.

  • Ações Condicionais: Se o usuário clicar em "Enviar", o sistema valida se os campos estão preenchidos corretamente e, se estiverem, salva as informações e direciona para a tela seguinte. Se o usuário clicar em "Voltar", o sistema retorna para a tela anterior sem salvar as informações.
