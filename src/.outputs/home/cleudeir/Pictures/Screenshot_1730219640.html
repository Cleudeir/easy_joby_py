Descrição da tela

Tela principal do aplicativo para clientes, exibindo serviços solicitados e finalizados.

Cartões com Indicadores:

Nome Tipo Validação Observação
Número Texto - Número de identificação do serviço.
Local Texto - Local do serviço.
Data Data Formato DD/MM/AAAA - HH:mm Data e hora do serviço.
Serviço Texto - Descrição do serviço.
Status Texto - Status atual do serviço.
Aprovar Botão - Solicita aprovação do deslocamento do profissional para o serviço (apenas para serviços com status "Buscando profissional").
Solicitar Novo Serviço Botão - Permite solicitar um novo serviço.

Regra de Negócio:

  • Lógica de Processamento: A tela exibe os serviços solicitados e finalizados do cliente. Para cada serviço, são apresentadas informações como número, local, data, serviço, e status. Se o status do serviço for "Buscando profissional", um botão "Aprovar" é exibido, permitindo ao cliente aprovar ou rejeitar o deslocamento do profissional. Um botão "Solicitar Novo Serviço" permite ao cliente solicitar um novo serviço.
  • Ações Condicionais: O botão "Aprovar" só é exibido se o status do serviço for "Buscando profissional". Ao clicar em "Aprovar", o sistema envia uma solicitação de aprovação do deslocamento para o sistema, atualizando o status do serviço. Ao clicar em "Solicitar Novo Serviço", o sistema redireciona o usuário para a tela de solicitação de novo serviço.
