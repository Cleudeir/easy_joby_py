Observação :

  • Os campos com o ícone de cadeado permitem que o valor seja travado para reutilização em futuras transferências, facilitando o preenchimento para valores que são frequentemente reutilizados.

Regra de Negócio:

  • Lógica de Processamento: O usuário preenche todos os campos obrigatórios do formulário. Ao clicar em "Próximo", os dados são validados. Se os dados forem válidos, o usuário prossegue para a próxima etapa do cadastro. Caso contrário, mensagens de erro são exibidas para os campos inválidos.

  • Ações Condicionais: Se todos os campos forem preenchidos corretamente, o botão "Próximo" fica habilitado. Caso contrário, o botão permanece desabilitado.
