Observação :

  • Os campos com o ícone de cadeado permitem que o valor seja travado para reutilização em futuras transferências, facilitando o preenchimento para valores que são frequentemente reutilizados.

Regra de Negócio:

  • Lógica de Processamento: O usuário preenche todos os campos obrigatórios do formulário. Ao clicar em "Próximo", o sistema valida os dados inseridos. Se todos os dados estiverem corretos, o usuário prossegue para a próxima etapa do cadastro. Caso contrário, o sistema exibe mensagens de erro indicando os campos que precisam ser corrigidos.

  • Ações Condicionais: Se o formulário for preenchido corretamente, o sistema redireciona o usuário para a próxima tela. Se houver erros de validação, o sistema permanece na tela de cadastro, exibindo mensagens de erro para cada campo inválido. O botão "Próximo" só será habilitado quando todos os campos obrigatórios forem preenchidos e validados corretamente.
