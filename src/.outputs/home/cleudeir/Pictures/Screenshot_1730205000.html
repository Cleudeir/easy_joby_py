Regra de Negócio:

  • Lógica de Processamento: O usuário preenche os campos do formulário com as informações do seu endereço. Ao clicar em "Próximo", os dados são validados e, se estiverem corretos, o usuário prossegue para a próxima etapa do cadastro. Ao clicar em "Voltar", o usuário retorna para a tela anterior.

  • Ações Condicionais: Se algum campo obrigatório estiver vazio ou com dados inválidos, o sistema exibirá uma mensagem de erro. A navegação entre telas é controlada pelos botões "Próximo" e "Voltar".
