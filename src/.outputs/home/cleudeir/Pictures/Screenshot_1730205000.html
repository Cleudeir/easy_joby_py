Descrição da tela

Tela de cadastro de endereço para o sistema UAU. O usuário preenche os campos com as informações do endereço e clica em "Próximo" para prosseguir.

Regra de Negócio:

  • Lógica de Processamento: O usuário preenche os campos do formulário com as informações do endereço. Ao clicar em "Próximo", os dados são validados. Se houver erros, o usuário é notificado. Caso os dados estejam corretos, o sistema prossegue para a próxima etapa do processo.

  • Ações Condicionais: O botão "Voltar" retorna o usuário para a tela anterior. O botão "Próximo" avança para a próxima tela somente se todos os campos obrigatórios estiverem preenchidos corretamente.
